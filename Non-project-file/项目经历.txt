心悦商城是一个基于 Spring Boot、Spring MVC、MyBatis 与 Thymeleaf 构建的在线商城项目，覆盖商品浏览、搜索筛选、购物车、订单、地址、收藏、客服、活动和后台管理等核心模块。我负责项目的整体业务实现与页面联调，重点完成了订单流程、库存扣减、优惠券使用、金币抵扣、订单状态流转和后台管理等功能，并通过 MyBatis XML 完成数据持久化设计，保证前后端数据一致性。

在项目优化过程中，我对购物车和立即购买流程做了库存边界校正，避免库存为 0 时仍可继续下单；同时梳理了订单状态、优惠类型等高频判断逻辑，减少魔法字符串带来的维护成本。针对主要业务链路，我补充了单元测试，覆盖购物车库存校验和订单辅助逻辑，提升了核心流程的稳定性。前端方面，我对统一布局、页脚样式和页面交互做了细节优化，提升了页面整体观感与可读性。项目还支持数据库自动建表、演示数据初始化、环境变量配置和 Windows 一键启动脚本，具备较完整的本地运行与演示能力。
